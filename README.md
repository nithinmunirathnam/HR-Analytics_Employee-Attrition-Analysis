# HR Attrition Analysis Dashboard

## Overview

This project focuses on analyzing employee attrition using SQL, Excel, and Power BI. The goal is to understand why employees are leaving the company and identify the key factors that influence attrition. The dashboard provides interactive visualizations that help HR teams make informed decisions to improve employee retention.


## Problem Statement

Employee attrition is a major challenge for organizations as it increases hiring costs and affects productivity. This project analyzes employee data to identify the main reasons behind employee turnover and provides insights that can help improve retention.


## Tools Used

- Microsoft Excel – Data Cleaning
- PostgreSQL – Data Analysis
- Power BI – Dashboard Development


## Project Workflow

1. Collected the employee dataset.
2. Cleaned the data using Microsoft Excel.
3. Imported the cleaned data into PostgreSQL.
4. Wrote SQL queries to answer business questions.
5. Connected PostgreSQL to Power BI.
6. Built interactive dashboards to visualize employee attrition.
7. Generated business insights and recommendations.

## Dashboard Pages

### Page 1 – HR Attrition Analysis
Provides an overview of employee attrition using KPIs such as Total Employees, Employees Left, Active Employees, Attrition Rate, Average Income, and Average Job Satisfaction. It also includes department-wise attrition, overtime analysis, gender distribution, and years at company.

### Page 2 – Root Cause Analysis
Focuses on identifying the main factors affecting employee attrition, including overtime, job satisfaction, work-life balance, income bands, and distance from home.

### Page 3 – Workforce Insights & Retention Strategies
Summarizes workforce distribution by job role, age group, and environment satisfaction. The page also highlights key insights and recommendations to improve employee retention.


## Key Insights

- Total Employees: **1,197**
- Employees Left: **278**
- Attrition Rate: **23%**
- Overtime is the strongest factor associated with employee attrition.
- Employees with lower job satisfaction and poor work-life balance are more likely to leave.
- Operations department recorded the highest number of employees leaving.


## Recommendation

- Reduce excessive overtime.
- Improve employee work-life balance.
- Increase employee engagement and job satisfaction.
- Focus retention efforts on departments with higher attrition.
